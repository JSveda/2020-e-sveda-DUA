Kroky binárního vyhledávání

Napište metodu s názvem pocetKrokuBinarnihoVyhledavani, která dostane v hlavičce jako parametr setřízené pole proměnných typu int dle velikosti a proměnnou typu int reprezentující hledané číslo a bude vracet proměnnou typu int. Metoda vrátí počet kroků, které program provedl, než našel hledané číslo. Pro nalezení čísla použijte algoritmus binárního vyhledávání.

Nápověda:

Počet kroků odpovídá počtu iterací while smyčky v algoritmu binárního vyhledávání.
