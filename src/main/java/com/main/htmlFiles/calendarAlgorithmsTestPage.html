Včerejší den

Napište metodu s názvem vcera, která dostane v hlavičce jako parametr 3 proměnné typu int reprezentující den, měsíc a rok a bude pole typu int, které bude mít 3 prvky reprezentující den, měsíc a rok (ve stejném pořadí). Algoritmus má za úkol vrátit takové pole, ve kterém bude uloženo datum včerejšího dnu.
