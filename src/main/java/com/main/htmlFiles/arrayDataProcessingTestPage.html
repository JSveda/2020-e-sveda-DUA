Posunutí prvků dozadu

Napište metodu s názvem posunDopredu, která dostane v hlavičce jako parametr pole typu int a proměnnou typu int značící, o kolik prvků se pole bude posouvat a bude vracet proměnnou typu int. Metoda posune prvky v poli o požadovaný počet míst a vrátí toto pole.

Nápověda:

Zamezte opakovanému posunu tak, že použijete kód ze stávající kapitoly: pocetKroku %= arr.length;
