EUKLEIDŮV ALGORITMUS

Tento algoritmus pochází původně ze starověkého Řecka od tehdejšího filozofa Eukleida. Přestože tento algoritmus uvedl ve svém díle Základy, není dodnes zcela jasné, kde tento algoritmus vynalezl. Zajímavý je tím, že je zřejmě nejstarším netriviálním algoritmem v historii lidstva.

Algoritmus je určený pro nalezení největšího společného dělitele (NSD) dvou přirozených čísel. Dále se dá také využít pro nalezení nejmenšího společného násobku (nsn) dvou přirozených čísel.
