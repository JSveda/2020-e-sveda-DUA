Zkuste pomocí jakéhokoli vývojového nástroje naprogramovat v jazyce Java program, který bude v samostatné třídě. Jeho zadání naleznete níže. Poté odevzdejte jeho verzi ve formátu nazevProgramu.java kliknutím na tlačítko "Vybrat soubor". Následně aplikace vyhodnotí, zda jste úkol splnil/a.
Pokud je Vaše řešení funkční, zobrazí se "zelená fajfka". V opačném případě se zobrazí červený křížek.

Minimum a maximum online algoritmem

Napište metodu s názvem minMaxZeVstupu, která dostane v hlavičce jako parametr objekt typu File a bude vracet pole typu int se dvěma prvky. První prvek pole bude reprezentovat zjištěné minimum a druhý maximum. Napište program, který bude hledat tyto hodnoty v textovém souboru, který dostane jako parametr.

Nápověda:

Pro čtení použijte objekt Scanner, který dostane v konstruktoru soubor, ze kterého budete chtít číst. Dále použijte jeho metodu nextInt(). Pro zjištění, jestli není konec souboru použije jeho metodu hasNext() s návratovou hodnotou boolean.
